# -------------- Rotinas do GIT
1. git status (verifique situação)
2. se vermelho -> git add .
3. se verde -> git commit -m "nome do commit"
4. se tudo ok -> git push (atualiza no github)
5. git clone (copia do repositorio e coloca na maquina)
6. pwd (mostra o caminho que vc está)
7. clear/cls (limpa a barra do terminal)

8.git clone endereco.git. (comando para copiar do github p local)
8.1.npm install (dentro da pasta do projeto p instalar node_modules)
